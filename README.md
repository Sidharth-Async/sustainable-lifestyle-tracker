This application is designed to help users track and manage their daily activities to promote a more sustainable and eco-friendly lifestyle. It provides tools to log environmentally conscious actions and visualize their impact.

‚ú® Features

The Sustainable Lifestyle Tracker offers simple, core functionalities to manage your sustainability journey:

    Activity Logging: Allows users to easily record daily sustainable actions (e.g., using public transport, recycling, reducing meat consumption).

    Impact Tracking: Calculates the basic environmental benefit or 'score' associated with the recorded actions.

    Progress Visualization: A dashboard feature to view trends and monitor personal progress over time.

üíª Technology Stack

This project is built using both a Java backend and a separate frontend interface.

    Backend: Java with the Spring Boot framework.

    Frontend: React.

    Build Tool: Apache Maven.

    Database: MySQL.

üöÄ Getting Started

Follow these steps to set up and run the project locally.

Prerequisites

You will need the following installed:

    Java Development Kit (JDK) 17+

    Apache Maven

    Git

Installation and Setup

    Clone the repository:
    Bash

git clone https://github.com/Sidharth-Async/sustainable-lifestyle-tracker.git
cd sustainable-lifestyle-tracker

Backend Setup (Java/Spring Boot):

    Navigate to the backend directory (usually the root directory for Spring Boot).

    Configure your database settings in the application.properties or application.yml file under src/main/resources.

    Build the project:
    Bash

./mvnw clean install

Run the application:
Bash

    java -jar target/sustainable-lifestyle-tracker-0.0.1-SNAPSHOT.jar

Frontend Setup :

    git clone https://github.com/Sidharth-Async/SLT-Frontend.git
    cd SLT-Frontend 
    npm install
    npm start

The backend usually starts on http://localhost:8080. Check your console output for the exact frontend URL (often http://localhost:3000).
